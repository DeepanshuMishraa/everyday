# Document Brief Format

```markdown
# What this document requires

## Identity
- Issuer:
- Document date / received date:
- Reference:
- Stated purpose:

## In plain language
<short summary>

## Actions and deadlines
| By when | Action | Documents needed | Submission method | Source on document |
|---|---|---|---|---|

## Gather
- [ ] Exact document or evidence

## Need to ask
- Question — why the text does not settle it

## Completion check
- [ ] Copies saved
- [ ] Submission confirmation saved
- [ ] Follow-up date recorded
```

Quote only the short wording needed to anchor a date, obligation, or ambiguity.
