# Offline Travel Brief Format

```markdown
# Travel day — <date>

## Timeline in local time
- Time zone — time — place — action — cutoff

## Connection risks
- Segment — required processes — available buffer — uncertainty

## Critical access
- Documents / medication / payment / keys / contacts location

## Verified contacts
- Operator/accommodation/insurance — independently sourced channel

## If disrupted
- Delay:
- Cancellation:
- Missed connection:
- Bag:
- Phone:
- Last transport:

## Recheck live
- Fact — authoritative source — when
```

Keep the brief short enough to screenshot or print on two pages.
