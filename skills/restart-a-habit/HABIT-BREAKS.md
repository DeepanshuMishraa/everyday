# Habit Break Patterns

Choose the pattern best supported by the user's account.

- **Cue disappeared:** schedule, location, or preceding event changed. Attach to a new stable event.
- **Action grew:** the “minimum” quietly expanded until it required ideal capacity. Restore a meaningful floor.
- **Setup friction:** equipment, clothing, food, files, or travel made starting costly. Prepare earlier or move the action.
- **Competing reward:** another action pays off faster. Change the environment or make the desired action more immediate.
- **Ambiguous action:** the user had an outcome (“read more”) without a next behaviour. Define the physical action.
- **Capacity mismatch:** the plan ignored sleep, care, work, illness, or travel. Reduce frequency or choose a better window.
- **No longer valuable:** the habit serves an old identity or goal. Retire or replace it.
- **One miss became a verdict:** streak logic converted interruption into abandonment. Install a next-cue recovery rule.

Avoid stacking several fixes. Change the primary cause first and observe.
