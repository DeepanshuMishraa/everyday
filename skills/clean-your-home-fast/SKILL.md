# Clean Your Home Fast

## Interaction

When missing information materially changes the outcome and the host exposes a dedicated user-input or question tool, use that tool. Ask no more than three short, related questions per call, then wait for the answers before asking the next batch.

Prefer selectable options when choices are concise and genuinely mutually exclusive; allow a free-form answer when needed. Do not present a long questionnaire in normal chat. If no native input tool is available, ask one concise blocking question at a time. For non-blocking gaps, state the assumption and continue.

Run a **reset**, not a deep clean. A reset restores safety, sanitation, and the next needed function inside a hard time box.

## Set the finish line

Ask for the time available, people helping, rooms affected, next use of the space, and any spills, food, pests, sharps, bodily fluids, chemicals, smoke, water, electrical, or structural hazards.

Translate the goal into at most three visible conditions—for example: “sink usable, toilet clean, clear path to seating.” Name what will remain undone.

## Work in this order

1. **Stop damage:** active spills, food left out, blocked exits, trip hazards.
2. **Remove:** rubbish, dishes, and laundry into their destinations or containers.
3. **Restore function:** sink, toilet, table, seating, walking path, or bed.
4. **Wipe the visible hygiene surfaces:** only those relevant to the next use.
5. **Close:** take rubbish out, return supplies, and stop when the timer ends.

Assign each helper one complete zone or outcome. Do not make one person carry the invisible job of repeatedly directing everyone else.

## Use containment deliberately

When sorting would consume the time box, put unrelated items in one named container with a review time. Containment is a deferred decision, not permission to hide food, moisture, hazards, or time-sensitive paperwork.

## Return

- the three-part finish line
- a minute-by-minute sequence
- one owner per zone or outcome
- a “do not do today” list
- one later task only if it prevents the reset from immediately unravelling

For mould, pests, chemicals, bodily fluids, sharps, fire, gas, electricity, major water ingress, or structural danger, replace ordinary cleaning instructions with safe isolation and appropriate local help.
