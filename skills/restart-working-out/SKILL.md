# Restart Working Out

Restore **repeatability** before fitness. Former ability is context, not the starting prescription.

## Confirm the kind of restart

Ask what the user did before, length and reason for the lapse, current daily activity, available days and time, preferred movement, equipment, and confidence.

Screen for injury, surgery, illness recovery, acute pain, concerning symptoms, pregnancy-specific needs, or clinician restrictions. Route those branches to qualified guidance; this skill does not provide rehabilitation or clearance.

## Choose an easy baseline

Select familiar movement the user is willing to repeat. Start below what motivation suggests: short enough and easy enough that finishing leaves meaningful reserve.

Use [TWO-WEEK-RESTART.md](TWO-WEEK-RESTART.md) to build the trial. Include a reduced version that preserves attendance on a difficult day.

## Control progression

Change only one variable at a time—frequency, duration, volume, or intensity—after the current level feels repeatable and recovery is ordinary for the user. A completed week does not automatically earn progression.

## Install the miss rule

After a missed session, continue with the next scheduled session or the reduced version. Do not repay missed work, double volume, or restart the calendar from zero.

Review completion, perceived effort, enjoyment, recovery, and concerning symptoms after two weeks. Continue, reduce, or progress using those observations.
