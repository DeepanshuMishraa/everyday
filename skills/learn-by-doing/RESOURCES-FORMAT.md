# Learning Resources Format

```markdown
# Resources for <learning goal>

## Core sources
- [Source title](url)
  Trust reason. Use for: specific topic or decision.

## Practice and feedback
- Resource, community, class, or tool
  Use for: the real-world feedback it provides.

## Gaps
- Knowledge or feedback source still missing
```

Prefer primary sources, recognised experts, direct documentation, and well-moderated communities. Annotate every entry. Remove weak or off-goal resources rather than accumulating links.
