# Learning Goal Format

```markdown
# Learning goal: <observable capability>

## Why
<real-world reason this matters>

## Success looks like
- The learner can <observable performance>
- The learner can <second performance>

## Starting point
- Prior knowledge and evidence
- Previous attempts

## Constraints
- Time, tools, access, safety, preferences

## Out of scope for now
- Adjacent topic deliberately deferred
```

Keep one coherent goal per workspace. Revise it when the learner's real goal changes, with their agreement.
