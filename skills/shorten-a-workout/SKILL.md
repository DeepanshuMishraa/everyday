# Shorten a Workout

Shorten the session; preserve the **training signal**. A twenty-minute workout is not a sixty-minute workout performed recklessly fast.

## 1. Screen the session

Ask about the planned workout, available minutes including setup, training experience, recent training, equipment, space, and current energy.

Ask directly about acute pain, injury, illness, chest pain, fainting, unusual breathlessness, pregnancy-specific needs, clinician restrictions, or a need for rehabilitation. When present, stop ordinary workout adaptation and direct the user to appropriate qualified guidance.

**Done when:** this is an ordinary time/equipment adaptation for a user who can safely choose to exercise.

## 2. Name the training signal

Choose one primary signal:

- strength practice on a key movement
- muscle-building volume for a target area
- aerobic base
- higher-intensity conditioning
- movement skill or technique
- mobility or recovery movement

If the user supplies a programme, preserve its stated intent. If the intent is unclear, infer cautiously from the main work and say so.

**Done when:** one sentence explains what must survive the cut.

## 3. Cut in the right order

Remove:

1. optional finishers
2. duplicate isolation or accessory work
3. extra sets beyond the minimum useful exposure
4. exercise variety that creates setup cost

Keep a brief specific warm-up and the work most directly tied to the signal. Do not compensate for less time with unfamiliar exercises, maximal attempts, or uncontrolled density.

Use [WORKOUT-CUTS.md](WORKOUT-CUTS.md) for signal-specific choices and [EFFORT-GUIDE.md](EFFORT-GUIDE.md) to describe intensity without pretending to prescribe medical or personalised coaching.

## 4. Return the runnable session

Give a clock-based plan:

1. **0:00–X:XX — Prepare:** warm-up and first work-set ramp.
2. **X:XX–Y:YY — Main work:** exercise, sets/reps or time, rest, effort ceiling.
3. **Y:YY–end — Optional only if time remains:** one supporting movement or easy finish.
4. **Stop if:** pain, dizziness, chest symptoms, loss of safe technique, or other concerning symptoms.

Include a quieter, lower-energy, or no-equipment substitution only when it preserves the same signal. End with the first setup action.
