# Grocery List Format

Consolidate the final plan into one list:

```markdown
## Produce
- Item — total quantity — used in Mon / Thu

## Protein and dairy
- ...

## Pantry and grains
- ...

## Frozen
- ...

## Check before buying
- Item — pantry stock uncertain
```

Combine duplicate quantities, subtract only inventory the user confirmed, and mark recipe quantities that depend on serving size. Put optional garnishes in a separate “only if wanted” group.
