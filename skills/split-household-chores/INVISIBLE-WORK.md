# Invisible Household Work

For each visible task, check the full loop:

- noticing the need
- deciding when it must happen
- checking supplies or prerequisites
- buying, booking, or coordinating
- doing the visible work
- checking the result
- remembering the next recurrence

Examples:

- “Cook dinner” includes choosing meals, checking food, shopping, timing, and leftovers.
- “Take the child to an appointment” includes booking, forms, records, transport, questions, and follow-up.
- “Clean the bathroom” includes noticing supplies, safe storage, frequency, towels, rubbish, and repairs.

Do not assign the visible task to one person while leaving another as permanent project manager.
