# Moving Plan Format

```markdown
# Home move — <move date>

## Fixed dates
- Date/time — event — owner — dependency

## Workstreams
### Packing
- [ ] Zone/outcome — owner — due date
### Utilities and addresses
- [ ] Service — stop/start date — confirmation
### Property handover
- [ ] Reading/photo/key action — owner

## Do-not-pack zone
- Documents
- Medication and care supplies
- Keys and access
- First-night bag

## Moving day roles
- Origin lead:
- Dependents/pets:
- Travel essentials:
- Destination lead:

## First night
- Sleep, hygiene, food, lighting, connectivity

## Delay fallback
- If access/vehicle/travel fails, first three actions
```
