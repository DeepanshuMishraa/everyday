# Care Handoff Format

```markdown
## Handoff — <date/time>

**From / to:**
**Current location and status:** factual, non-clinical description
**What happened since last handoff:**
**What is due next and when:**
**Supplies or documents:**
**Professional instructions to preserve:**
**Questions or changes to escalate:**
**Handoff accepted at:**
```

Do not infer a diagnosis or alter an instruction in the handoff.
