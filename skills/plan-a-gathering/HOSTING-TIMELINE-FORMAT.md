# Hosting Timeline Format

```markdown
# Gathering plan

**Guests arrive:** <date and time>
**Protected buffer begins:** <time>

## Two to seven days before
- [ ] Confirm guest count, allergies, access needs
- [ ] Buy or order durable items

## Day before
- [ ] Prepare items that improve overnight
- [ ] Set up seating, serving ware, coats, bathroom

## Day of — backward from arrival
- T−180: task — owner — equipment
- T−120: task — owner — equipment
- T−60: task — owner — equipment
- T−30: stop cooking that needs attention; host gets ready
- T−15: lights, music, water, first-arrival position

## Fallbacks
- If <failure>, then <simpler action>
```

Keep the protected buffer visible. If the timeline overruns it, remove or buy something; do not silently spend the buffer.
