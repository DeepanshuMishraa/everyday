# Exposure Response

Choose the branch matching what happened:

## Credentials or codes shared

Use the independently verified service, change the password, end other sessions, review recovery methods, enable strong multi-factor authentication, and inspect recent activity. Protect the email account first when it controls recovery.

## Payment made or card details shared

Contact the financial institution through a verified channel immediately, explain the exact transaction, and follow its fraud process. Preserve receipts and messages.

## File opened or remote access granted

Disconnect the device from sensitive activity, end remote access if safe, use trusted device-security guidance, and contact qualified technical support when compromise is possible. Do not ask the user to run unknown cleanup tools.

## Identity information shared

Record exactly what was disclosed and use current official identity-protection guidance for the user's country.

## Nothing shared

Preserve evidence, report through the platform or appropriate local channel, block the sender, and delete only after evidence is saved.
